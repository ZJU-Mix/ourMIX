# 专业必修课
## 1. 电路与模拟电子技术
### 1.1 课程介绍
学分最大的课（5.5），由两门3学分长学期课程合并而来，内容上相当于是做了加法。

前半个学期主要是电路元件及其性质、基尔霍夫定律及电路分析法、交流电电路及其分析。需要熟练掌握电路（包括直流电路和交流电路）的分析、计算方法，并能够根据需要运用定理和定律、结论进行解题。

后半个学期主要是二极管、三极管、简单放大电路、多级放大电路、运算放大电路、功率放大电路和波形发生电路。需要熟练掌握放大电路的分析方法的基础上，了解常见的电路结构，并能够进行相应的计算

需要基础：简单物理电路基础、常微分方程。
### 1.2 授课老师
=== "孙盾"

    （模电最难抢的教学班，没有之一）。

    孙盾老师教授电学基础课已经三十多年，上课是富有激情的。她的课件里面有大量的例题，并且讲解很细致，偶尔走神几分钟也能靠例题了解知识点。老师也非常关心同学，即使极少数人没到课或连续几周没交作业也很着急学生落下（虽然她的班上考勤不占分，而且作业补交不扣分）。不过也因为盾妈讲课较细，她的班里的进度往往要慢于其他教学班，而且有笔记检查部分，可能有几周的作业量突增。

    日常答疑的时候经常吸引其他班同学来。钉钉问问题甚至会直接给你打钉钉电话 ~~（胆小者慎操作）~~ ，并且可能会详细告诉你知识点在哪个PPT的那一页。给分上不用多说。盾妈班是有组队大作业的，也相当于是送点分。

### 1.3 考核形式
- 30%平时分，包括作业、小测、考勤、慕课（孙盾班没有考勤和小测，有一次组队大作业）
- 30%期中考试，包括整个电路分析理论部分
- 40%期末考试，包括整个模拟电子技术部分
### 1.4 课程资源/学习建议
虽然被连续多届学长冠以“魔电”称号——学分高、内容多、上课听不懂、下课不会做。但是请各位务必放轻松，这门课并没有想象中那么难，它绝对比各位的数学通识课简单。对待这样一门课程最大的忌讳就是“畏惧”。

个人比较推荐孙盾老师的课，或者如果学习比较吃力的话智云孙盾老师也可以，说不定会有醍醐灌顶的感觉。日常的作业多多思考，虽然教材有参考答案，但是题目并不难，一定要自主练习，考试中的中档题在一定程度是基于课后习题，有时会将作业简单改编后作为考试题目。在此基础上，建议大家在重点难点内容之后采用笔记整理的方式回归知识点本身，可以很好地巩固。而且基础打扎实，老师的授课也就容易理解，后面的课程会越学越简单。

最后，笔者认为整个计算量最大的部分是交流电分析部分（因为涉及复数运算），如果可能的话推荐使用卡西欧计算器。考前推荐过一遍老师的课件（能拿到孙盾老师的最好），并通过历年卷练手（虽然是古早历年卷，但是题型和考点分布都是类似的），对于各种电路模型务必熟练掌握。

## 2. 电路与模拟电子技术实验
### 2.1 课程介绍
模电的配套实验课，主要结合理论课进行11次左右实验，包括电路原理实验和模拟电子技术实验，并撰写实验报告。最后一次会有自己焊接电路并完成测试的任务。整体上课程难度低，但实验报告比较费时。由于老师不一致，常常出现理论课进度慢于实验课。

需要基础：模电理论基础
### 2.2 授课老师
=== "孙盾"

=== "干于"

    老师整体上也是正常水平。中规中矩，个人感觉算是中位数水平吧，念PPT比较多。有时候任务不明确，需要去多问问才知道具体要求做什么。
    
    在作业上，实验报告的要求没有普物实验那么高，但是似乎比较注重公式的存在（雾）；课前老师会简单讲评上一次的报告，可以据此改进报告。

    给分的话，受限于五级制，也没法再高了。

=== "周晶"

### 2.3 考核形式
- 60%平时分，包括考勤、实验报告（电子版，其中有1-2次需要打印出来）等
- 40%期末，在实验室机考，包括客观题和实验操作题

考核采用五级制评价，一般以班级前30%为优秀，其余为良好。
### 2.4 课程资源/学习建议
实验报告按照普物实验的标准弄，然后风格上稍微迎合一下老师的要求就行。期末考试个人感觉有一些一锤定音的味道（大多实验报告的给分比较接近，导致期末的影响较大），想拿高分的话需要准备一下，对一些理论知识、实验操作等要具备一定的熟悉度。
## 3. 自动控制理论
### 3.1 课程介绍
据传是控院最好的两门课之一，介绍连续系统的控制理论，包括传递函数、状态空间模型、稳定性判断、稳态误差、动态响应和系统补偿器设计等，知识点难度适中。自控这门课是对复变课程的超级大量的实际运用，从初期传递函数所需的拉普拉斯变换到后期的留数定理等，几乎所有内容都是建立在复数域之上。

需要基础：复变函数与拉普拉斯变换、线性代数、常微分方程
### 3.2 授课老师
=== "赵豫红"

    赵豫红老师是控院教学办主管科研训练的老师，个人认为是行政老师里讲课最好的之一。

    赵老师讲课有一种不紧不慢的感觉，很平和，每个知识点都会尽量讲透，这也导致她的课缺乏重点，不过总体还好。平时会有几次小测，几乎不提前通知，一般会快下课的时候做一道题，如果时间不够会留作当晚截止的作业。（小测的时候翻课件查公式都是可以的。）

    给分方面还可以，或许有点捞但不是很捞，属于正常水平，高分还是得自己真实力。

### 3.3 考核方式
- 40-60%平时分：包括作业、小测、考勤等
- 60-40%期末考试：半开卷，可携带一张手写或打印A4纸。一个大知识点一道大题，一共7道。21级及之前有小题，但是因为普遍卷子写不完，22级取消了小题。

平时分和期末考试的具体比例会参考是否有期中考试、期末考试的答题情况等共同决定。
### 3.4 课程资源/学习建议
平时学好，作业并不难，尽量自己完成；教学组会公布作业答案，如果有错误的话可以参考并修正。考前尽量每个内容都过一下，比如本身是后面《现代控制理论》重点的状态空间模型和能观、能控标准型也会考到。考试前虽然可以带A4，但是仍然建议自己复习一下。一晚速通是会寄的（血泪教训）。

98上有学长们的A4纸，可以选择一个最合适的打印；或者自己写，可以很好地顺便复习。自控的A4内容明显比隔壁嵌入式少，可以不用那么挤。

考试最好带上卡西欧991CN这一类能协助解方程的，可能一些方程次数高且系数极大，计算器协助计算可以加快做题速度。
## 4. 嵌入式系统
### 4.1 课程介绍
嵌入式系统是一门硬件为主的课程，主要包括数电基础和汇编语言、8051及其编程、STM32及其编程。但是，不得不说这门课的容量真的不是一门课程能搞定的。

需要基础：C程序设计及实验（或程序设计与算法基础）等C语言基础
### 4.2 授课老师
=== "王酉/李光"

    李光老师是王酉老师的导师，一般不上课。王酉老师是主要做球形机器人方向的，因此嵌入式系统对他来说算本业了。

    老师上课还可以，可以看出来他是懂的，但是嵌入式系统的课程内容一言难尽。王酉老师班的特点是三周速通数字电路，几乎很少有直接实验指导的8051和STM32部分，很多都需要自己摸索。老师会讲到要用的所有知识点，但是从理论到实践是需要自学的部分。在实物作业上，王酉老师会要求在实验课前完成，课上主要是验收和改进，可能会导致连续多个晚上熬夜甚至通宵。

    王酉老师有一两次点名和小测，平时作业不多，两次实物实验。给分上正常偏好，但是王酉老师出分极慢（考后3个月才出分的老师了解一下，笑哭）。


### 4.3 考核形式
=== "王酉/李光"

    - 10%平时分：包括考勤、小测、平时作业
    - 30%实验成绩，包括软件实验（上机实验）和实物实验
    - 60%期末考试：半开卷，仅能携带一张手写A4（不能打印），包括判断、选择、程序填空、设计题等。
### 4.4 课程资源/学习建议
虽然这门课是4.0学分的课，但是除非你能力出众，否则建议不要在这门课上耗费太多时间。平常作业最好几个人一起合作讨论讨论，遇到编程bug的时候，笔者比较推荐请计算机学院学过数逻和计组的同学来协助，找bug效率高两倍不止。（硬件的bug永远比你想的要玄学。）

期末复习主要是靠王酉老师最后一节课的复习课，建议连带智云、课件和98上之前学长的A4纸一起抄（98上不少前人的资源，包括笔者的），笔最好细一点（因为字需要写很小），最好三色笔一起记。这个A4如果认真写下来可能要用接近十个小时，请务必给予足够的时间，提前开始并仔细打磨。其他不抄的内容也推荐顺便看看，其他的复习就不需要太多了。22级考核内容与王酉老师说的略带出入（例如老师说SR触发器不用记但是最后考了一个选择），所以整个复习课PPT的东西务必有印象。
